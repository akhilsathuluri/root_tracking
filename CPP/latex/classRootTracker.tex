The D\+M\+Tracker uses the Davindenkos\textquotesingle{} integration method to find the solutions satisfied by the constrain equations. The output is the values of the unknown variables at each tracking step. The problem of tracking is solved as an initial value problem using the first order derivative form of the constraint equations. This function accepts functions as arguments using the {\ttfamily C++11} style functional library. This method by default uses the Explicit Euler integration scheme and is the only supported scheme currently.
